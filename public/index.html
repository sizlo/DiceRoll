Dice Roll

A slack bot to perform dice rolls.

Commands

Roll

Roll a dice with Y sides X number of times.

/roll [XdY]

Random User

Pick a random user from the current conversation

/randomuser

For private conversations the @dice_roll user will need to be added to the conversation

Pick a random user from a usergroup

/randomuser @someusergroup

Privacy Policy

This bot does not log, store or share any user data.
